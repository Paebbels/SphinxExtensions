SphinxExtensions Documentation
##############################

Extensions for the Sphinx documentation tool.


Installation/Updates
********************

Installation using PIP
======================

.. code-block:: bash

   pip3 install SphinxExtensions


Updating using PIP
==================

.. code-block:: bash

   pip3 install -U SphinxExtensions
